그리고 확장자 .java 붙여야 함

class Ex01{} =>
 - class Ex01
 - class 는 자바에서 파일로 저장하는 단위
 - Ex01 는 class 파일의 이름
-  class 파일의 이름은 무조건 첫글자는 대문자
-  API는 회사에서 제공하는 class 파일을 모아놓은 것
-  무조건 파일이름이나 클래스 이름은 같아야 한다
-  저장이름 = 클래스이름 = 파일이름

System.out.println("자바의 꿈")
-  웹브라우저에서 즐겨찾기에서 API 호출
-  왼쪽 상단 패키지창에서 All Classes를 클릭하면
-  왼쪽 하단 클래스창에서 모든 클래스가 나타남
-  ctrl + f => 찾기 대화상자 호출
-  해당 클래스를 입력하여 찾는다. (System)
-  in    => 표준입력장치 키보드를 뜻하는 말
-  out   => 표준출력장치 모니터를 뜻하는 말
-  err   => 표준오류출력장치 모니터를 뜻하는 말

XXXXXX([내용]) : 자바에서 ()는 무조건 메소드이다. 
		메소드란 기능, 동작, 작동 실행을 의미한다
		메소드는 클래스 안에 존재한다.

main()는 JVM이 해석하고 실행할 때 제일 처음 호출해서 사용하는 메소드로 해당 프로그램에서 오직 하나만 존재 (시작위치)
없어도 안되고 여려개 있어도 안됨

데이터 타입 (자료형) : 처리할 수 있는 자료의 형태
 = boolean(참, 거짓), 문자, 숫자(정수, 실수)

데이터를 저장하는 저장공간(기억공간)을 변수, 상수
변수 : 데이터를 변경할 수 있음 (한번 집어넣으면 값이 변할 수 있음, 변수의 이름은 모든 글자를 소문자로 사용, 가장 최근의 정보 하나만 기억)
상수 : 데이터를 변경할 수 없음 (한번 집어넣으면 값이 안 변함 X, 상수의 이름은 모든 글자를 대문자로 사용)

변수나 상수를 호출하면 가지고 있는 데이터가 나온다.

Ex. 시스템 변수 편집 -> JAVA_HOME이란 상수 이름을 호출하면 변수 값이 나옴
